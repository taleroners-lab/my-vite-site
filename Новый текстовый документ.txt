🔥 Отличный вопрос — это прям следующий уровень уже

Ты хочешь:
👉 менять сайт на ПК → и понимать что нажимать, чтобы он обновился

Сделаем тебе **простую шпаргалку (как у профи)** 👇

---

# 🚀 ТВОЙ ПРОЦЕСС ОБНОВЛЕНИЯ САЙТА

## 📂 1. Вносишь изменения

На ПК:

* меняешь код (`src/`, `index.html`, стили и т.д.)
* добавляешь картинки

---

## 🧪 2. Проверяешь локально (желательно)

```bash
npm run dev
```

👉 открываешь:

```
http://localhost:5173
```

---

## 🚀 3. Отправляешь на сайт (САМОЕ ГЛАВНОЕ)

ВСЕГДА делай это:

```bash
git add .
git commit -m "что изменил"
git push
```

---

## ⚡ 4. Что происходит автоматически

* Vercel сам видит push
* сам делает build
* сам обновляет сайт

👉 тебе ничего больше делать не нужно

---

# ⏳ Время обновления

* обычно: **20–40 секунд**

---

# 🔍 Как проверить

Открываешь:

```
https://eliseyh.art
```

👉 если не видно изменений:

```text
Ctrl + F5
```

(жёсткое обновление)
